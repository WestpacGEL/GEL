# FormCheck

Checkbox and radio inputs are provided by the FormCheck component.

FormCheck items are block-level by default; inline layout is available via the `inline` prop. FormCheck toggles can sit on the left (default) of the label text, but can be configured to render on the right via the `flipped` prop.

## Type

Use the `type` prop to render FormCheck as a `checkbox` or `radio` control.

<Playground>
	<FormCheck type="checkbox" name="example-checkbox">
		<Option value="1">Default checkbox 1</Option>
		<Option value="2">Default checkbox 2</Option>
		<Option value="2" disabled>
			Disabled checkbox 3
		</Option>
	</FormCheck>
</Playground>
