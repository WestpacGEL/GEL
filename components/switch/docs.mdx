# Switch

These are very simple toggle switches ideal for use in settings or preference screens. The switch can be blank or contain simple toggle text labels (no/yes, off/on etc).

<Playground>
	<Switch name="ex-default" id="ex-default">
		Turn notifications
	</Switch>
</Playground>

## Label text

Form switches require label text in order to meet accessibility requirements. You can visually hide this text by providing a `srOnlyText` prop if necessary.

<Playground>
	<Switch name="ex-text" id="ex-text" toggleText={['Yes', 'No']}>
		Enable notifications
	</Switch>
	<Switch name="ex-text-sronlytext" id="ex-text-sronlytext" toggleText={['Yes', 'No']} srOnlyText>
		Enable notifications
	</Switch>
</Playground>

## Sizes

Form switches come in four sizes.

<Playground>
	<Switch name="ex-small" id="ex-small" size="small">
		Small
	</Switch>
	<Switch name="ex-medium" id="ex-medium" size="medium">
		Medium
	</Switch>
	<Switch name="ex-large" id="ex-large" size="large">
		Large
	</Switch>
	<Switch name="ex-xlarge" id="ex-xlarge" size="xlarge">
		Extra large
	</Switch>
</Playground>

Form switches can be sized responsively by passing an array of size values for each breakpoint.

<Playground>
	<Switch name="ex-responsive" id="ex-responsive" size={['small', 'medium', 'large', 'xlarge']}>
		Responsive form switch
	</Switch>
</Playground>

## Flip

Flip mode renders the toggle on the left of the label text.

<Playground>
	<Switch name="ex-flip" id="ex-flip" flip>
		Flip mode
	</Switch>
</Playground>

## Block

Block mode extends the width so it spans the full width of its parent.

<Playground>
	<Switch name="ex-block" id="ex-block" block>
		Block mode
	</Switch>
</Playground>
