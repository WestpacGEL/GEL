# Button Group

Group a series of buttons together on a single line to form a ButtonGroup toggle. These function much like a radio input set.

ButtonGroup components inherit Button appearance, sizing and icon options, and can render in block mode similar to block buttons.

<Playground>
	<ButtonGroup name="example-buttongroup">
		<Button value="left">Left</Button>
		<Button value="middle">Middle</Button>
		<Button value="right">Right</Button>
	</ButtonGroup>
</Playground>
