# Accessibility helpers

Components used to provide features for digital accessibility compliance.

## SrOnly

The SrOnly component visibly hides an element to all devices except screen readers.

<Playground>
	<p>The text below this message is visibily hidden...</p>
	<SrOnly>This is screen reader only text</SrOnly>
</Playground>
