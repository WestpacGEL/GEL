# Accessibility helpers

Components used to provide features for digital accessibility compliance.

## SrOnly

The SrOnly component hides an element to all devices except screen readers.

<Playground>
	<p>The text below this message is visibly hidden...</p>
	<SrOnly>This is screen reader only text</SrOnly>
</Playground>

## SrSkipLink

The SrSkipLink component hides an element to all devices except screen readers. It links via an anchor hash to an area of content on the page and becomes visible and fixed to the top of the viewport when focussed.

There are generally one or more SrSkipLink components on a page, inserted as necessary to meet accessibility compliance requirements.

<Playground>
	<p>The link below this message is visibly hidden until focussed...</p>
	<SrSkipLink href="#content">This is screen reader only text (visible when foccused)</SrSkipLink>
	<br />
	<div id="content">
		<p>
			This is example content, linked to by the SrSkipLink component above. Sit sint irure do eu.
			Non aliquip voluptate et nisi est voluptate in aliquip. Tempor ea est et velit anim incididunt
			qui ipsum anim id. Irure exercitation adipisicing velit minim ea aute esse elit amet minim in
			minim cillum. Cupidatat aliqua eiusmod ipsum occaecat proident exercitation.
		</p>
	</div>
</Playground>
