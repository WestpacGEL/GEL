# Label

Labels are styled elements to highlight words or links.

<Playground>
	<Label>Default</Label>{' '}

    <Label appearance="primary">Primary</Label>{' '}
    <Label appearance="hero">Hero</Label>{' '}
    <Label appearance="neutral">Neutral</Label>{' '}
    <Label appearance="faint">Faint</Label>{' '}

    <Label appearance="success">Success</Label>{' '}
    <Label appearance="information">Information</Label>{' '}
    <Label appearance="warning">Warning</Label>{' '}
    <Label appearance="danger">Danger</Label>

</Playground>

## Label as a link

Simply pass a `href` prop to render Label as a link.

<Playground>
	<Label href="#0">Default</Label>{' '}

    <Label href="#0" appearance="primary">Primary</Label>{' '}
    <Label href="#0" appearance="hero">Hero</Label>{' '}
    <Label href="#0" appearance="neutral">Neutral</Label>{' '}
    <Label href="#0" appearance="faint">Faint</Label>{' '}

    <Label href="#0" appearance="success">Success</Label>{' '}
    <Label href="#0" appearance="information">Information</Label>{' '}
    <Label href="#0" appearance="warning">Warning</Label>{' '}
    <Label href="#0" appearance="danger">Danger</Label>

</Playground>
